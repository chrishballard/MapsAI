**Export `fetchAttributes(params: { googleAccountId: string; locationName: string }): Promise<{ attributes: GBPAttribute[]; error?: string }>`:**

Define a TypeScript interface for the response:
```typescript
interface GBPAttributeValue {
  attributeId: string;
  valueType: "BOOL" | "ENUM" | "REPEATED_ENUM" | "URL";
  values?: unknown[];
  repeatedEnumValue?: {
    setValues?: string[];
    unsetValues?: string[];
  };
  uriValues?: Array<{ uri: string }>;
}

interface GBPAttributeMetadata {
  attributeId: string;
  valueType: "BOOL" | "ENUM" | "REPEATED_ENUM" | "URL";
  displayName?: string;
  groupDisplayName?: string;
  valueMetadata?: Array<{
    value: string;
    displayName?: string;
  }>;
}

export interface GBPAttribute {
  attributeId: string;
  displayName: string;
  groupDisplayName: string;
  valueType: "BOOL" | "ENUM" | "REPEATED_ENUM" | "URL";
  currentValue: unknown;
  valueMetadata?: Array<{ value: string; displayName?: string }>;
}
```

Implementation:
- Create OAuth client via `createGoogleClient(googleAccountId)`
- Make TWO requests in parallel with `Promise.all`:
  1. GET `https://mybusinessbusinessinformation.googleapis.com/v1/${locationName}/attributes` -- returns current attribute values set on this location
  2. GET `https://mybusinessbusinessinformation.googleapis.com/v1/attributes` with query param `categoryName` extracted from the location's category, OR simply GET `https://mybusinessbusinessinformation.googleapis.com/v1/${locationName}/attributes` which returns both available and set attributes. Note: The GBP API returns available attributes with their metadata (valueType, groupDisplayName, possible enum values) alongside current values. If a single endpoint returns both, use that.
- If the GBP API returns attributes grouped with metadata: merge current values with available attributes metadata
- For each attribute in the response, build a `GBPAttribute` object:
  - `attributeId`: the attribute ID string
  - `displayName`: from metadata, or derive from attributeId using the existing `titleCase` helper
  - `groupDisplayName`: from metadata, or default to "Other"
  - `valueType`: BOOL, ENUM, REPEATED_ENUM, or URL
  - `currentValue`: for BOOL: `values[0]` (boolean or null), for ENUM: `values[0]` (string or null), for REPEATED_ENUM: `{ setValues, unsetValues }`, for URL: `uriValues[0]?.uri` (string or null)
  - `valueMetadata`: for ENUM/REPEATED_ENUM types, include the possible values array
- Return `{ attributes: [...] }` on success
- On error: return `{ attributes: [], error: message }` -- do NOT throw
- If the response has zero attributes, return `{ attributes: [] }` (no error -- some categories simply have none)

**Export `pushAttributesToGBP(params: { googleAccountId: string; locationName: string; attributes: Array<{ attributeId: string; valueType: string; values?: unknown[]; repeatedEnumValue?: { setValues?: string[]; unsetValues?: string[] }; uriValues?: Array<{ uri: string }> }> }): Promise<{ success: boolean; error?: string }>`:**

Implementation:
- Create OAuth client via `createGoogleClient(googleAccountId)`
- Build the request body. The GBP attributes PATCH expects the attributes array in the format the API returns:
  ```typescript
  {
    attributes: params.attributes.map(attr => {
      const base: Record<string, unknown> = {
        attributeId: attr.attributeId,
        valueType: attr.valueType,
      };
      if (attr.valueType === "BOOL" || attr.valueType === "ENUM") {
        base.values = attr.values;
      } else if (attr.valueType === "REPEATED_ENUM") {
        base.repeatedEnumValue = attr.repeatedEnumValue;
      } else if (attr.valueType === "URL") {
        base.uriValues = attr.uriValues;
      }
      return base;
    })
  }
  ```
- PATCH `https://mybusinessbusinessinformation.googleapis.com/v1/${locationName}/attributes`
- Body: the built attributes object
- On success: return `{ success: true }`
- On error: catch, extract error message, return `{ success: false, error: message }`
- Do NOT throw -- caller handles failures gracefully (same pattern as pushDescriptionToGBP and pushServicesToGBP)
